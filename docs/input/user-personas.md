# ターゲットユーザー

## メインペルソナ

### 基本情報
- 名前（仮）：田中 翔太
- 年齢：28歳
- 職業：IT企業の営業職
- 生活環境：都内一人暮らし、平日は残業多め、休日は疲れて家にいることが多い

### 1日の流れ
- 7:30 起床、支度
- 8:30 通勤（電車40分、スマホでSNSやニュースを見る）
- 9:00〜20:00 仕事（昼休みもスマホ）
- 21:00 帰宅、夕食
- 22:00〜23:30 ベッドでスマホ（TikTok、Instagram、YouTubeをダラダラ見る）
- 23:30 寝なきゃと思いつつ、なかなかスマホを置けない
- 0:00 就寝

### 課題・ペイン
- 旅行が好きだが、まとまった休みが取れず年に1〜2回しか行けない
- 仕事終わりにリフレッシュしたいが、結局ショート動画で1時間以上消える
- SNSを見た後、充実感はなく「また時間を無駄にした」という罪悪感がある
- 寝る前に目が冴えてしまい、睡眠の質が悪い

### 現在の解決方法
- Instagramで旅行系アカウントをフォローして写真を見る（→ 余計にスマホ時間が増える）
- Spotifyの「カフェBGM」「波の音」プレイリスト（→ たまに聴くが習慣化していない）
- 旅行の計画を立てる（→ 行けないので逆にストレス）

### 求めているもの
- 短時間で完結するリラックス体験
- 視覚を使わない、目を閉じられるコンテンツ
- 「その場にいる感覚」を得られるリアルな音

### このプロダクトを使う瞬間
- 寝る前、ベッドに入ってから就寝までの時間
- 目を閉じて30秒、どこかの波の音や街の音を聴いてから眠りにつく

### 情報収集の方法
- Twitter（X）で話題のアプリを知る
- App Storeのランキング
- 友人の口コミ
