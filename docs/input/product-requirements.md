# プロダクト要件

## 1. プロダクト概要

### プロダクト形態
スマートフォンアプリ（iOS / Android）

### 対応プラットフォーム
- iOS（優先）
- Android

### 利用環境
- 寝る前、ベッドの中でイヤホンをつけて利用
- 暗い部屋での使用が多い（ダークUIが必須）
- Wi-Fi環境下での利用が中心

## 2. コアバリュー

### このプロダクトの核となる価値
目を閉じて30秒、誰かがその場で録った音で旅先の空気を感じる「余白の時間」

### Aha Moment
地図上のピンをタップして目を閉じた瞬間、まるでその場にいるかのような環境音が流れてきて「あ、これいい」と感じる瞬間

## 3. 主要機能

### 必須機能（Must Have）
- 世界地図の表示と音声スポットのピン表示
- ピンタップ → 投稿情報表示 → 30秒音声再生
- 音声の録音（環境音・生活音、30秒）
- 位置情報付きでの音声投稿
- ユーザー登録・ログイン

### 推奨機能（Should Have）
- いいね（保存）機能
- 保存した音の一覧画面
- 投稿一覧（自分の投稿履歴）

### あれば嬉しい機能（Nice to Have）
- エリア・国ごとのフィルター
- 高音質再生（プレミアム）
- オフライン保存（プレミアム）
- 季節・時間帯タグ
- 人気スポットのハイライト

## 4. 非機能要件

### パフォーマンス
- 地図の表示：2秒以内
- 音声の再生開始：タップから1秒以内
- 投稿完了：録音終了から5秒以内

### セキュリティ
- ユーザー認証（メール / SNSログイン）
- 音声データの安全な保管
- 不適切コンテンツの通報機能

### スケーラビリティ
- MVP時点：数百ユーザー
- 拡大期：数万ユーザー規模の音声配信に耐えうる設計
