# ユーザーフロー

## 1. オンボーディングフロー

### 新規登録
1. アプリを起動する
2. コンセプト紹介画面（1〜2枚）：「目を閉じて30秒、音で旅しよう」
3. アカウント登録（メールアドレス or SNSログイン）
4. 地図画面（ホーム）に遷移

### 初回利用
1. 地図画面が表示される（世界地図にピンが表示されている）
2. 気になるピンをタップ
3. 投稿情報（場所名、投稿者名、録音日時）が表示される
4. 再生ボタンをタップ
5. 30秒間、音声が再生される（画面に「目を閉じて」の案内）
6. 30秒で自動停止

## 2. メインフロー

### 音を聴く（コア体験）
**目的：** 寝る前に30秒の余白を得る

**フロー：**
1. アプリを開く → 地図画面が表示される
2. 地図を操作（ズーム、スクロール）して気になるエリアを探す
3. ピンをタップする
4. 投稿情報が表示される（場所名、投稿者名、録音日時）
5. 再生ボタンをタップ
6. 30秒間、環境音が再生される
7. 30秒経過で自動停止
8. 別のピンを聴くか、アプリを閉じる

**成功条件：** ユーザーが1つ以上の音を最後まで（30秒間）聴き終える

---

### 音を投稿する
**目的：** 旅先や日常で録った環境音を共有する

**フロー：**
1. 投稿ボタン（＋ボタン）をタップ
2. 録音画面に遷移
3. 録音ボタンをタップして録音開始
4. 30秒で自動停止（または手動停止）
5. プレビュー再生で確認
6. 場所名を入力（位置情報は自動取得）
7. 投稿ボタンをタップ
8. 地図上にピンが追加される

**成功条件：** 音声が位置情報付きで正常に投稿され、地図上にピンとして表示される

## 3. その他のフロー

### ログイン
1. アプリを起動
2. メールアドレス & パスワード or SNSログイン
3. 地図画面に遷移

### いいね（保存）[MVP後]
1. 音声再生中 or 再生後に保存ボタンをタップ
2. 保存一覧に追加される

### 保存した音の再生 [MVP後]
1. 保存一覧画面を開く
2. リストから音を選択
3. 30秒再生
