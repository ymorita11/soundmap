## 機能詳細

### 音声再生
**概要：** 地図上のピンをタップし、30秒間の環境音・生活音を再生する

**ユーザーストーリー：**
「旅行に行けない社会人として、寝る前に世界中の音を聴きたい。なぜなら、目を閉じて30秒の余白を得てリラックスしたいから。」

**入力：**
- ピンのタップ
- 再生ボタンのタップ

**出力：**
- 30秒間の音声再生
- 再生中のUI表示（残り時間など）

**受け入れ条件：**
- [ ] ピンタップで投稿情報が表示される
- [ ] 再生ボタンタップから1秒以内に音声が再生開始される
- [ ] 30秒経過で自動停止する
- [ ] 再生中に音量調整が可能

---

### 音声録音・投稿
**概要：** スマホのマイクで環境音を30秒録音し、位置情報付きで投稿する

**ユーザーストーリー：**
「旅先にいるユーザーとして、今聴こえている音を録音して共有したい。なぜなら、この場所の空気感を他の人にも届けたいから。」

**入力：**
- マイクからの音声入力（30秒）
- 場所名（テキスト）
- 位置情報（自動取得）

**出力：**
- 地図上に新しいピンが追加される
- 他のユーザーがその音を再生できるようになる

**受け入れ条件：**
- [ ] 録音ボタンタップで録音が開始される
- [ ] 30秒で自動停止する
- [ ] プレビュー再生で録音内容を確認できる
- [ ] 録り直しが可能
- [ ] 位置情報が自動で取得される
- [ ] 場所名を入力できる
- [ ] 投稿完了後、地図上にピンが表示される
