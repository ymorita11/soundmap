# SoundMap — 音で旅する地図SNS

## 概要
旅先や日常の環境音・生活音を録音・投稿し、世界地図上で共有する音声特化型SNS。目を閉じて30秒、音だけで旅先の空気を感じる「余白の時間」を届ける。

## 解決する課題
- 旅行に行きたくても時間がない社会人が、旅行気分を味わえない
- SNSやショート動画による情報疲れ・視覚的なノイズに消耗している
- 日常に「目を閉じて30秒リラックスする余白」がない

## ターゲットユーザー
時間がなく旅行に行けない社会人。日常の情報過多に疲れ、視覚に頼らないリフレッシュを求めている人。

## 主要機能
- 世界地図上に音声スポットをピン表示
- ピンをタップして30秒の環境音・生活音を再生
- 旅先や日常で録音した音を位置情報付きで投稿
- いいね（保存）で気に入った音をストック
- ユーザー登録・ログイン
